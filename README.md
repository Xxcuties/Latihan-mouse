# 🐭✨ Latihan Mouse ✨🐭

## 🎀 Deskripsi
**Latihan Mouse** adalah aplikasi kecil berbasis **HTML, CSS, dan JavaScript** untuk melatih keterampilan dasar penggunaan mouse.  
Target utamanya adalah anak-anak SD yang masih sering bingung membedakan **klik kiri** 🖱️ dan **klik kanan** 🖱️.  
Dengan pendekatan berbentuk game interaktif, aplikasi ini bisa menjadi media belajar tambahan yang menyenangkan 💖.

👉 Untuk guru dan pengajar, silakan langsung mencoba di sini:  
[🌈 Latihan Mouse Demo](https://xxcuties.github.io/Latihan-mouse/)

---

## 🌸 Latar Belakang
Banyak guru menemukan bahwa murid SD masih kesulitan membedakan klik kanan dan kiri.  
Aplikasi ini dibuat sebagai solusi edukatif yang ringan, imut, dan mudah dijalankan di browser.  
Guru atau pengembang bisa memanfaatkannya sebagai **alat bantu pembelajaran praktis** di kelas maupun di rumah 🏫🏡.

---

## 🍬 Fitur
- ⏱️ **Timer interaktif** dengan indikator warna (biru → oranye → merah).
- ✅❌ **Feedback visual**: benar, salah, atau timeout ⏰.
- 📊 **Sistem skor**:
  - Benar: +10  
  - Salah: -5  
  - Timeout: -3
- 🎯 **Evaluasi akhir** dengan pesan motivasi imut:
  - *“Luar biasa!”* 🌟  
  - *“Lumayan!”* 🌼  
  - *“Ayo latihan lagi!”* 🌈
